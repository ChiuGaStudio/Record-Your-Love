這是一個 SwiftUI iOS 應用程式，可用於掃描和管理條碼。它讓你能快速使用相機擷取條碼，在儲存時加入額外資訊，並可將資料以 QR Code 方式匯出。

📷

條碼掃描－使用相機掃描 EAN8、EAN13、Code128、Code39 和 QR Code

1️⃣

單筆掃描－一次擷取一個條碼

📋

條碼列表－檢視所有儲存的條碼，可依號碼、描述或分類搜尋並刪除不需要的項目

⌨️

手動輸入－若無法掃描時手動新增條碼

✏️

條碼詳細畫面－可編輯照片、描述、分類、價格等資訊，並分享截圖

🔲

QR Code 產生－建立包含所有條碼的 QR Code 並分享

⚙️

設定－查看簡短指南並分享應用程式連結
